# Contact Roles

A **Contact Role** defines the functional capacity in which a contact represents themselves relative to an assigned object. Example roles include: `Technical Support`, `Sales Vendor`, `Emergency Contact`, or `Billing Administrator`.

## Attributes

| Field | Description | Type | Required |
| --- | --- | --- | --- |
| **Description** | Optional notes detailing role expectations. | Text | No |
| **Name** | Unique name of the role (e.g. `Technical Support`). | String | Yes |
| **Slug** | URL-safe name representation. | Slug | Yes |
