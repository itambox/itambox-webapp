# Contacts

A **Contact** represents a real-world individual or external organization associated with physical infrastructure, manufacturer support channels, software licensing vendors, or internal team escalations.

## Attributes

| Field | Description | Type | Required |
| --- | --- | --- | --- |
| **Comments** | The comments of the contact. | Text | No |
| **Description** | Optional notes describing the contact context. | Text | No |
| **Email** | Primary contact email address. | Email | No |
| **Name** | The full name of the contact (e.g. `John Smith` or `Supplier Sales Team`). | String | Yes |
| **Phone** | Primary telephone number. | String | No |
| **Title** | The job title or function of the contact. | String | No |
| **Web URL** | Website page link. | URL | No |
